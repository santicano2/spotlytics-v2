# Spoti-lytics

Spot-lytics is an app that let's you see your latest Spotify stats. It includes two categories, top tracks and top artists, which you can filter by time. (4 weeks, 6 months or 1 year).

## Stack

- Next.js
- TypeScript
- TailwindCSS
- ShadCn
- NextAuth

## API

- Spotify API

### Install packages

```shell
npm install
```

### Setup .env

```js
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
```

### Start app

```shell
npm run dev
```
